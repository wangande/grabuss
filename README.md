#飞机疲劳实验中的异常数据分析
#采用改良后的格拉布斯法则
#支持样本量10-100

参考文摘：http://doc.mbalib.com/view/0de98de6e62d06d19a8bced723838ebd.html

测试数据：啤酒浑浊度数据
原始数据：0.532 0.543 0.557 0.546 0.656 0.691 0.546 0.527 0.565 0.556 0.582 0.551 0.556
数据量：13 

#运行步骤
1、进入程序目录
2、运行make，生成aircraft_fatigue程序
3、./aircraft_fatigue
4、根据提示输入
5、计算结果
6、结果保存
7、可以从相关文件中查看历史运行结果
